# タスク: シェアリンクから同じ問題に挑戦できる対戦機能

## ユーザー体験
1. ユーザーAが問題を解いて結果をシェア
2. シェアリンクを開いたユーザーBが「この問題に挑戦する」を押す
3. ユーザーAと同じ日本語の問題文が表示される
4. ユーザーBが回答すると、添削結果がユーザーAのスコアと並べて表示される

## 結果ページの変更（/share/[id]）
現在:
- ユーザーAの結果を表示するだけ

変更後:
- 「この問題に挑戦する」ボタンを追加
- ボタンを押すと、同じ問題文で回答入力画面に遷移
- 回答後、ユーザーBの結果とユーザーAの結果を並べて比較表示
  - 左右 or 上下でスコア・回答・模範解答を比較
- ユーザーBも自分の結果を更にシェアできる（対戦チェーン）
